Taming OpenClaw: Security Analysis and Mitigation of Autonomous LLM Agent Threats
Xinhao Deng, Yixiang Zhang, Jiaqing Wu et al. 2026-03-12
核心贡献
本文针对自主LLM智能体（OpenClaw）提出五层生命周期安全框架，系统分析提示注入、技能供应链污染、记忆中毒和意图漂移等复合威胁，并提出缓解策略。
This paper introduces a five-layer lifecycle security framework for autonomous LLM agents (OpenClaw) and systematically analyzes compound threats including prompt injection, skill supply chain contamination, memory poisoning, and intent drift, along with mitigation strategies.
重要性与意义 / Significance
随着自主LLM智能体在高权限真实环境中的广泛部署，本文提供了首批针对此类系统的结构化生命周期威胁分类框架，为安全智能体系统的研究者和实践者提供了实用路线图。
As autonomous LLM agents are increasingly deployed in high-privilege, real-world environments, understanding their attack surface is critical for safe deployment. This paper provides one of the first structured, lifecycle-oriented threat taxonomies for such agents, offering a practical roadmap for both researchers and practitioners building secure agentic systems. The compound-threat perspective goes beyond single-vector analyses common in prior work.
科研问题 / Research Question
OpenClaw等自主LLM智能体具有高执行权限和即时消息集成，其攻击面远超传统LLM部署。本文研究其整个运行生命周期中的系统性复合安全威胁及有效缓解方法。
Autonomous LLM agents like OpenClaw operate with high execution privileges and tight instant-messaging integration, dramatically expanding their attack surface compared to traditional LLM deployments. The paper asks: what are the systematic, compound security threats across the full operational lifecycle of such agents, and how can they be effectively mitigated?
先前工作为何不够 / Why Prior Work Falls Short
现有LLM安全防御多为单点防御，仅针对单一威胁向量，未考虑跨阶段复合攻击。早期框架主要关注单轮交互，忽视了有状态、长周期、工具调用场景下的记忆持久化和技能供应链攻击。
Existing defenses against LLM threats (e.g., prompt injection filters, sandboxing) are largely point-based, addressing individual threat vectors in isolation rather than compound, cross-stage attacks. Prior security analyses focus predominantly on single-turn LLM interactions and do not account for the long-horizon, stateful, tool-using nature of autonomous agents. Memory persistence and skill/plugin supply chains introduce attack vectors that earlier frameworks largely ignore.
解决方案 / Solution
作者提出涵盖初始化、输入、推理、决策和执行五层的生命周期安全框架，将各威胁类型（如技能供应链污染、间接提示注入、记忆中毒、意图漂移）精确映射至对应层次，并通过OpenClaw案例研究验证威胁的实际可利用性，同时提出针对复合攻击的整体缓解策略。
The authors propose a five-layer lifecycle-oriented security framework covering: initialization, input, inference, decision, and execution stages of agent operation. Each layer is mapped to specific threat classes — for example, skill supply chain contamination at initialization, indirect prompt injection at input, memory poisoning at inference, and intent drift at decision/execution. Detailed case studies on OpenClaw are used to demonstrate real exploitability of these threats. The paper also analyzes why existing point-based defenses fail against compound, multi-stage attack chains and proposes more holistic mitigation strategies.
主要结论 / Key Results
OpenClaw案例研究证实四类主要威胁均可在真实部署中被利用。跨生命周期阶段的复合攻击（如记忆中毒叠加提示注入）会突破现有单点防御，暴露出严重的安全缺口。
Case studies on OpenClaw confirm that all four major threat classes (indirect prompt injection, skill supply chain contamination, memory poisoning, intent drift) are practically exploitable in real agent deployments. The analysis reveals that existing defenses leave critical gaps when threats compound across lifecycle stages — for instance, a poisoned memory state can amplify the impact of a subsequent prompt injection. Specific quantitative metrics are not prominently reported in the abstract, but the qualitative severity assessment underscores the prevalence of these vulnerabilities.
值得关注的讨论点 / Discussion Points
  • 复合威胁视角：真实攻击往往链式跨越多个阶段（如供应链污染→记忆持久化→意图漂移），单点防御不足，未来应优先研究跨层检测机制。
  • 评估范围局限：分析主要基于OpenClaw，五层框架对ReAct、AutoGPT等不同记忆与工具调用架构的泛化能力有待验证。
  • 缓解方案可行性：运行时意图监控、技能来源验证等整体缓解措施可能引入延迟与复杂性，其对自适应攻击者的实际效果尚需基准测试验证。
  • Compound vs. point threats: The paper's key insight is that real-world attacks chain multiple stages (e.g., supply chain poisoning → memory persistence → intent drift), making isolated defenses insufficient — future work should prioritize cross-layer detection mechanisms.
  • Scope of evaluation: The analysis is grounded in OpenClaw specifically; it remains an open question how well the five-layer framework generalizes to other agent architectures (e.g., ReAct, AutoGPT) with different memory and tool-use designs.
  • Mitigation feasibility: Proposed holistic mitigations (e.g., runtime intent monitoring, skill provenance verification) may introduce latency and complexity trade-offs in production systems, and their empirical effectiveness against adaptive adversaries has yet to be benchmarked.

Jailbreak Scaling Laws for Large Language Models: Polynomial-Exponential Crossover
Indranil Halder, Annesya Banerjee, Cengiz Pehlevan 2026-03-11
核心贡献
本文利用自旋玻璃统计力学框架，推导出LLM越狱成功率从多项式到指数的缩放规律，揭示长注入提示导致攻击成功率指数级增长。
This paper derives polynomial-to-exponential scaling laws for LLM jailbreak success rates using a spin-glass statistical mechanics framework, showing that longer injected prompts cause exponential growth in attack success.
重要性与意义 / Significance
本文首次为越狱攻击随推理计算量的缩放提供严格理论基础，对推理时扩展日益普及的安全研究具有重要意义。自旋玻璃框架为分析LLM安全失效提供了超越纯实验观察的理论工具，有助于从业者评估和预测对抗性风险。
This work provides the first rigorous theoretical foundation for understanding how jailbreak attacks scale with inference-time compute, a critical safety concern as test-time scaling becomes mainstream. The spin-glass framework offers a principled lens for analyzing LLM safety failures beyond empirical observation. Practitioners can use these scaling laws to anticipate and bound worst-case adversarial risk under prompt-injection attacks.
科研问题 / Research Question
对齐安全的LLM越狱攻击成功率如何随推理时采样数量缩放，背后机制是什么？随着推理时重复采样成本极低，经验安全基准可能严重低估实际最坏情况下的脆弱性。
How does the success rate of adversarial jailbreak attacks on safety-aligned LLMs scale with the number of inference-time samples, and what mechanism drives this scaling? Understanding this matters because best-of-N sampling and repeated querying are cheap at inference time, making empirical safety benchmarks potentially misleading about true worst-case vulnerability.
先前工作为何不够 / Why Prior Work Falls Short
已有工作主要聚焦于离散提示优化（如GCG）或经验性最优N次评估，缺乏对攻击成功率随样本数缩放函数形式的理论解释。现有安全评估通常报告单次或固定预算成功率，忽略重复采样下的渐近行为，也无理论框架将LLM生成分布与对抗缩放联系起来。
Prior work on adversarial attacks largely focuses on discrete prompt optimization (e.g., GCG) or empirical best-of-N evaluations, without explaining the functional form of how attack success scales with samples. Existing safety evaluations typically report single-query or fixed-budget success rates, missing the asymptotic behavior under repeated sampling. No prior theoretical framework connects LLM generation distributions to adversarial scaling through a principled probabilistic model.
解决方案 / Solution
作者将LLM生成建模为自旋玻璃能量景观上的Gibbs测度采样（处于复制对称破缺RSB区域），输出空间聚类为多个亚稳态，部分低能簇被标记为"不安全"。提示注入被建模为将生成偏向不安全簇中心的外部磁场：短注入（弱磁场）产生多项式缩放，长注入（强磁场）产生指数缩放。上述行为经解析推导并在多个安全对齐LLM上实验验证。
The authors model LLM token generation as samples from a Gibbs measure over a spin-glass energy landscape operating in a replica-symmetry-breaking (RSB) regime, where the output space is clustered into many metastable states. A subset of low-energy clusters is designated "unsafe," and prompt injection is modeled as an external magnetic field biasing generation toward unsafe cluster centers. Short injections (weak field) produce polynomial scaling of attack success rate with sample count N, while long injections (strong field) produce exponential scaling. These behaviors are derived analytically and validated empirically across multiple safety-aligned LLMs.
主要结论 / Key Results
实验表明，无提示注入时攻击成功率随采样数呈多项式增长（增速慢）；加入对抗性提示注入后，成功率转变为指数增长，即即使安全性较高的模型在足够多采样下也能被可靠越狱。理论预测与多个模型的实测缩放曲线吻合，验证了自旋玻璃框架的有效性。
Empirically, without prompt injection, attack success rate grows polynomially with the number of inference-time samples (slow growth). With adversarial prompt injection, success rate transitions to exponential growth, meaning even modestly safe models can be reliably jailbroken with enough samples. The theoretical predictions match observed scaling curves across tested models, validating the spin-glass generative framework.

Systematic Scaling Analysis of Jailbreak Attacks in Large Language Models
Xiangwen Wang, Ananth Balashankar, Varun Chandrasekaran 2026-03-11
核心贡献
本文首次提出针对大语言模型越狱攻击的缩放定律框架，通过FLOPs衡量四类攻击范式的成功率与计算成本之间的关系。
This paper introduces the first scaling-law framework for LLM jailbreak attacks, measuring attack success against attacker compute (FLOPs) across four attack paradigms to reveal efficiency trade-offs and predictable scaling behaviors.
重要性与意义 / Significance
本文填补了越狱攻击效率统一基准的空白，为防御者预测攻击能力增长提供了理论依据，也为攻击研究者提供了系统性的评估方法论。
This work provides the first unified, compute-normalized benchmark for comparing jailbreak attack efficiency, filling a critical gap in LLM safety research. By establishing scaling laws, it enables defenders to anticipate attack capability growth and allocate defense resources more strategically. The framework also offers attack researchers a principled methodology for evaluating and comparing new methods.
科研问题 / Research Question
越狱攻击的成功率如何随攻击者计算预算增长而变化？缺乏统一框架导致无法公平比较不同攻击方法，也无法预测攻击随资源增加的危险程度。
How does jailbreak attack success scale with attacker compute budget across different attack methods, model families, and harm categories? This matters because without a principled scaling framework, it is impossible to fairly compare attack methods or predict how much more dangerous attacks become as adversaries invest more resources.
先前工作为何不够 / Why Prior Work Falls Short
此前研究孤立评估各类越狱攻击，缺乏统一的计算量基准，方法间的比较结论不可靠，也未有研究将缩放定律引入越狱攻击分析。
Prior jailbreak studies evaluate attacks in isolation using inconsistent metrics and budgets, making cross-method comparisons unreliable. Existing work lacks a shared axis for attacker effort, so it is unclear whether one method outperforms another due to algorithmic superiority or simply more compute. No previous study has applied scaling-law analysis to the adversarial jailbreaking setting.
解决方案 / Solution
将所有越狱攻击统一映射到FLOPs轴，评估四类范式（优化、自优化提示、采样选择、遗传优化），用饱和指数函数拟合成功率曲线，提取渐近成功率等效率指标，并支持攻击性能外推预测。
The authors treat each jailbreak attack as a compute-bounded optimization and map all methods onto a shared FLOPs axis to enable fair comparison. They evaluate four paradigms: optimization-based attacks (e.g., GCG), self-refinement prompting, sampling-based selection, and genetic optimization, across multiple model families and scales. A saturating exponential function is fitted to FLOPs–success trajectories to extract interpretable efficiency summaries such as asymptotic success rate and compute-to-threshold metrics. This framework yields both empirical comparisons and a predictive tool for extrapolating attack performance under larger budgets.
主要结论 / Key Results
基于提示的范式（自优化与采样）在单位FLOPs成功率上优于GCG等优化类攻击；成功率曲线呈饱和指数趋势，存在方法特定的渐近上限；效率排名在不同模型族和危害类型上总体一致。
Prompting-based paradigms (self-refinement and sampling) are consistently the most compute-efficient jailbreak methods, achieving higher attack success scores per FLOP compared to optimization-based approaches like GCG. Scaling laws fit well with a saturating exponential, indicating diminishing returns as attacker budget grows, with attack success plateauing at method-specific asymptotes. Efficiency rankings are broadly consistent across model families and harm types, though absolute success rates vary.
值得关注的讨论点 / Discussion Points
  • 饱和指数规律表明无限增加计算量不能无限提升攻击成功率，每种方法存在天花板，对威胁建模有重要意义。
  • 提示类攻击计算效率最高，说明低成本对抗策略仍是更现实的威胁，防御优先级应向此类攻击倾斜。
  • 以FLOPs为统一预算指标可能忽略API限速、token费用等现实约束，未来研究应纳入时钟时间和货币成本等维度。
  • The saturating exponential scaling behavior implies that indefinitely increasing attacker compute does not guarantee unbounded success — each method has a ceiling, which is an important insight for threat modeling.
  • The finding that prompting-based attacks dominate in compute efficiency suggests that simple, low-cost adversarial strategies remain a more practical threat than expensive gradient-based optimization, which has implications for which attack surfaces defenders should prioritize.
  • The framework's reliance on FLOPs as a universal budget metric may not fully capture real-world attacker constraints such as API rate limits, token costs, or human-in-the-loop effort, suggesting future work should extend the axis to include wall-clock time and monetary cost.

MACPO: Weak-to-Strong Alignment via Multi-Agent Contrastive Preference Optimization
Yougang Lyu, Lingyong Yan, Zihan Wang et al. 2024-10-10 ICLR
核心贡献
MACPO提出多智能体对比偏好优化框架，通过迭代增强正向行为、惩罚负向行为，实现弱教师与强学生LLM之间的相互对齐。
MACPO introduces a multi-agent contrastive preference optimization framework that enables weak teacher LLMs and strong student LLMs to mutually improve alignment through iterative positive behavior augmentation and negative behavior penalization.
重要性与意义 / Significance
随着LLM在特定任务上超越人类，弱到强对齐问题成为安全部署AI的关键瓶颈。MACPO为该领域提供了实用框架，对可扩展监督与AI安全研究具有直接意义。
As LLMs increasingly surpass human capabilities on specific tasks, the weak-to-strong alignment problem becomes a critical bottleneck for safe AI deployment. MACPO offers a practical framework that addresses this underexplored setting, potentially guiding how future superhuman models can still be steered by weaker human supervisors. This work has direct implications for scalable oversight and AI safety research communities.
科研问题 / Research Question
本文研究如何仅凭弱教师LLM生成的监督信号有效对齐更强的学生LLM。当模型能力超越人类时，人工监督信号的质量上限不足，使此问题尤为关键。
The paper tackles the weak-to-strong alignment problem: how to effectively align a strong student LLM using only weak supervision signals generated by a less capable teacher LLM. This matters because as models approach or exceed human-level performance, human-generated supervision becomes an unreliable quality ceiling for alignment.
先前工作为何不够 / Why Prior Work Falls Short
现有RLHF、DPO等方法针对强到弱场景设计，自对齐方法依赖模型自身输出，均无法应对学生模型系统性超越教师的场景。
Existing alignment methods such as RLHF and DPO are designed for strong-to-weak settings, where supervisors are at least as capable as the models being aligned. Self-alignment approaches rely on the model's own outputs for supervision, which fails to introduce new behavioral improvements beyond the model's current capability boundary. Neither paradigm is equipped to handle scenarios where the student model systematically outperforms its teacher.
解决方案 / Solution
MACPO通过对比偏好优化构建多智能体协作框架，设计互向正向行为增强策略，使弱教师向强学生学习、强学生从教师已知正向行为中获益，同时惩罚双方的熟悉负向行为，迭代提升对齐质量上限。
MACPO proposes a multi-agent framework where weak teachers and strong students collaboratively improve each other through contrastive preference optimization. A mutual positive behavior augmentation strategy is devised so each agent learns from the other's strengths — the teacher learns better behaviors from the student, while the student is guided toward behaviors unfamiliar to it but known to the teacher. Negative behavior penalization further ensures that familiar but undesirable behaviors are suppressed in both agents. This iterative loop progressively raises the alignment quality ceiling beyond what either agent could achieve alone.
主要结论 / Key Results
MACPO在多个基准测试中持续优于现有对齐基线，互向学习机制带来的对齐质量提升超过强到弱及自对齐方法，在标准对齐评估指标上优于DPO及自博弈变体。
MACPO demonstrates consistent improvements over existing alignment baselines in the weak-to-strong setting across multiple benchmarks. The framework shows that mutual learning between agents yields higher alignment quality than either strong-to-weak or self-alignment approaches applied to the same setting. Specific numerical gains are reported across standard alignment evaluation metrics, with MACPO outperforming competitive baselines including standard DPO and self-play variants.
值得关注的讨论点 / Discussion Points

Prior dexterous manipulation research largely focuses on rigid tools (e.g., hammers, pens), where tool geometry remains fixed and does not interact with task success in a configuration-dependent way. Existing hierarchical RL methods for manipulation typically decompose arm and hand control but do not model tool affordance states that change with the tool's own articulation. Point cloud-based affordance estimation has been explored for rigid objects but rarely applied to estimate the functional state of a deformable or articulated tool during in-hand use.
解决方案 / Solution

The framework uses two policy layers: a low-level policy trained to actuate the dexterous hand to reach target tool configurations (e.g., specific tweezer opening angles), and a high-level policy that sets the tool's goal configuration and controls the robotic arm to position it for picking. A point cloud encoder, trained on synthetic data, maps observed tool geometry to affordance states—specifically, which opening angle supports grasping a given object size—providing the high-level policy with perception-grounded goal proposals. This decomposition separates the finger-level dexterity problem from the arm-level task planning problem, reducing the overall learning complexity.
